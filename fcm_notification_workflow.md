# โครงสร้างและขั้นตอนการทำระบบแจ้งเตือน (FCM Push Notification)
**โปรเจค:** KMUTNB Shuttle Tracker
**เป้าหมาย:** ระบบตรวจเช็คเวลาเดินรถและส่งการแจ้งเตือน (Push Notification) ไปยังแอปพลิเคชันของ "พนักงานขับรถ" ล่วงหน้า 15 นาที ก่อนถึงรอบเดินรถ

---

## บริบทของฐานข้อมูล (Database Context)
* **Firestore -> `users` collection:** มีฟิลด์ `role` (เช่น "driver") และฟิลด์ `fcm_token` (เก็บรหัสประจำเครื่องของคนขับ)
* **Firestore -> `schedules` collection:** มีฟิลด์ `start_time` (เวลาเริ่มรอบรถ เช่น "08:00") และ `bus_id`
* **Firestore -> `buses` collection:** มีฟิลด์ `bus_id` และ `driver_id` (ผูกรถกับคนขับ)

---

## ขั้นตอนการทำงานที่ต้องการให้ AI ช่วยเขียนโค้ด (Tasks for AI)

### ส่วนที่ 1: ฝั่ง Mobile App (Flutter - สำหรับแอปคนขับ)
**โจทย์ให้ AI ช่วยเขียนโค้ด:**
1.  ขอโค้ดสำหรับตั้งค่า `firebase_messaging` ใน Flutter
2.  ขอฟังก์ชันสำหรับการขอสิทธิ์ (Request Permission) แจ้งเตือนจากผู้ใช้งาน
3.  ขอฟังก์ชันสำหรับดึงค่า `fcm_token` จากเครื่องมือถือ และนำค่านี้ไปอัปเดตหรือบันทึกลงใน Cloud Firestore ที่คอลเลกชัน `users` (ใน Document ของคนขับที่ล็อกอินอยู่)
4.  ขอโค้ดสำหรับรับการแจ้งเตือน (Listen to messages) ทั้งตอนที่แอปเปิดอยู่ (Foreground) และปิดอยู่ (Background)

### ส่วนที่ 2: การเตรียมสิทธิ์ฝั่งเซิร์ฟเวอร์ (Firebase Service Account)
**โจทย์ให้ AI ช่วยอธิบาย:**
* ขอขั้นตอนสั้นๆ ในการเข้าไปที่ Firebase Console เพื่อดาวน์โหลดไฟล์ `service-account-file.json` สำหรับนำมาใช้ยืนยันตัวตนบนเซิร์ฟเวอร์ PHP (เพื่อใช้ส่ง FCM ผ่าน HTTP v1 API)

### ส่วนที่ 3: ฝั่ง Web Server (PHP Backend)
**โจทย์ให้ AI ช่วยเขียนโค้ด:**
ระบบนี้จะใช้ไฟล์ PHP ทำหน้าที่เป็นตัวเช็คเวลาและยิงแจ้งเตือน (ทำงานบนเซิร์ฟเวอร์หลังบ้าน) 
ขอโค้ดไฟล์ `check_schedule.php` โดยมี Logic ดังนี้:
1.  ดึงข้อมูลรอบรถจาก Firestore คอลเลกชัน `schedules`
2.  ดึงเวลาปัจจุบันของเซิร์ฟเวอร์ (Timezone: Asia/Bangkok)
3.  เปรียบเทียบว่ามีรอบรถไหนที่ `start_time` ตรงกับ **เวลาปัจจุบัน + 15 นาที** พอดีหรือไม่
4.  หากมีรอบรถที่ตรงเงื่อนไข ให้ดึง `bus_id` ไปค้นหา `driver_id` ในคอลเลกชัน `buses`
5.  นำ `driver_id` ไปค้นหา `fcm_token` ในคอลเลกชัน `users`
6.  ใช้ cURL ใน PHP ส่ง Request ไปที่ Firebase Cloud Messaging (FCM HTTP v1 API) เพื่อยิง Push Notification แจ้งเตือนคนขับคนนั้น โดยให้มีหัวข้อว่า "เตรียมตัวออกรถ!" และข้อความ "รถของคุณมีรอบวิ่งในอีก 15 นาที" (ต้องมีโค้ดตัวอย่างการสร้าง Access Token จากไฟล์ `service-account-file.json` ด้วย)

### ส่วนที่ 4: การตั้งเวลาทำงานอัตโนมัติ (Cron Job)
**โจทย์ให้ AI ช่วยอธิบาย:**
* ขอวิธีการนำไฟล์ `check_schedule.php` ไปแขวนไว้บนเซิร์ฟเวอร์ และวิธีการใช้เว็บฟรีอย่าง `cron-job.org` ในการตั้งค่าให้ระบบเรียกใช้งานไฟล์ PHP นี้ทุกๆ 1 นาที
